## Key Concepts

- **Statistics**: science of collecting, organizing, analyzing, and interpreting data
- **Population** vs **Sample**; **Parameter** vs **Statistic**
- **Descriptive** vs **Inferential** statistics
- Data types: **Qualitative** vs **Quantitative**
- Levels of measurement: **Nominal** < **Ordinal** < **Interval** < **Ratio**
- Data collection: observational study, experiment, simulation, survey
- Experimental design: confounding variables, blinding, randomization, replication
- Sampling: simple random, stratified, cluster, systematic, convenience
