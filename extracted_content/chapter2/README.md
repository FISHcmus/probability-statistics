## Key Concepts

- **Frequency distributions**: classes, midpoints, relative/cumulative frequencies
- **Graphs**: histograms, frequency polygons, ogives, stem-and-leaf plots, dot plots, pie charts, Pareto charts, scatter plots, time series
- **Central tendency**: mean ($\bar{x}$, $\mu$), median, mode, weighted mean, grouped data mean
- **Variation**: range, variance ($s^2$, $\sigma^2$), standard deviation ($s$, $\sigma$), Empirical Rule, Chebychev's Theorem
- **Position**: quartiles, IQR, percentiles, z-scores ($z = \frac{x - \mu}{\sigma}$), box-and-whisker plots
